# Synergetics Canopy

A free, open-source home for Buckminster Fuller's *Synergetics* — all 1,468 pages, brought forward into the light of the modern web. It takes **[R.W. Gray's Synergetics on the Web](https://www.rwgrayprojects.com/synergetics/synergetics.html)** and gives it new life: searchable, linkable, explorable. No accounts. No paywalls.

Instant search across all 1,468 pages. AI assistant trained on the complete text. Paragraph-level anchor links, one-click copy. Open source.

The project lives at [synergetics.pages.dev](https://synergetics.pages.dev).
